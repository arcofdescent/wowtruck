# Developer Tools List

- **GitHub Team or GitLab equivalent** - Source control, pull requests, code reviews, and release history - Free + paid tiers.
- **Jira, Linear, or GitHub Projects** - Backlog, sprint, bug, and release tracking - Free + paid tiers.
- **ChatGPT Business accounts for all developers** - AI assistance for architecture, coding, testing, documentation, and debugging - Paid.
- **VS Code** - Primary code editor for frontend, backend, scripts, and configuration files - Free.
- **PostgreSQL** - Main relational database for users, orders, fleet, payments, and audit records - Free software; infrastructure/hosting may be paid.
- **Redis** - Cache and short-lived data store for OTPs, sessions, queues, idempotency, and rate limits - Free software; infrastructure/hosting may be paid.
- **DBeaver** - Database client for browsing schemas, running SQL, and inspecting data - Free Community Edition; paid editions available.
- **Docker and Docker Compose** - Local development containers for database, cache, queues, and application services - Free for many uses; paid plans may apply for Docker Desktop in some commercial contexts.
- **Postman or Bruno** - API testing and shared request collections for backend and third-party integrations - Postman has free + paid tiers; Bruno is free/open-source with paid options.
- **Figma** - UI/UX design, wireframes, prototypes, and developer handoff - Free + paid tiers.
- **GitHub Actions or GitLab CI** - Automated builds, tests, checks, and deployments - Free usage limits + paid tiers.
- **Playwright** - Browser-based end-to-end testing for admin and customer web workflows - Free/open-source.
- **Sentry** - Application error tracking for backend, frontend, and mobile apps - Free + paid tiers.
- **Grafana/Prometheus or Grafana Cloud** - Metrics dashboards and monitoring for applications and infrastructure - Prometheus and self-hosted Grafana are free/open-source; Grafana Cloud has free + paid tiers.
- **AWS, Azure, or GCP account with separate dev, staging, and production environments** - Cloud hosting for applications, databases, storage, networking, and managed services - Paid cloud usage; limited free tiers may apply.
- **S3-compatible object storage** - Storage for documents, PODs, invoices, KYC files, and generated reports - Paid in cloud; MinIO is free/open-source if self-hosted.
- **SMS, email, payment gateway, maps, and e-KYC sandbox accounts** - External service sandboxes for notifications, payments, routing, and compliance verification - Usually free sandbox access; production usage is paid.
- **GitHub Dependabot, Trivy, and Gitleaks for basic security checks** - Dependency, container, and secret scanning in the development workflow - Free/open-source or included with GitHub.
