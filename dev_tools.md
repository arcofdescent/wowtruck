# Developer Tools List

## 11. Recommended Minimum Starter Stack

- GitHub Team or GitLab equivalent.
- Jira, Linear, or GitHub Projects.
- ChatGPT Business accounts for all developers.
- VS Code.
- PostgreSQL.
- Redis.
- DBeaver.
- Docker and Docker Compose.
- Postman or Bruno.
- Figma.
- GitHub Actions or GitLab CI.
- Playwright.
- Sentry.
- Grafana/Prometheus or Grafana Cloud.
- AWS, Azure, or GCP account with separate dev, staging, and production environments.
- S3-compatible object storage.
- SMS, email, payment gateway, maps, and e-KYC sandbox accounts.
- GitHub Dependabot, Trivy, and Gitleaks for basic security checks.
